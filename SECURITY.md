# Security Policy

## Reporting a Vulnerability

As this is a private bachelor thesis repository, security vulnerabilities are unlikely to affect a wider user base. However, if you discover a security issue (e.g., leaked credentials, misconfiguration), please report it immediately.

### Reporting Process

Please report sensitive information directly to the repository owner via email or private message. **Do not create a public issue.**

## Supported Versions

This project is a research thesis and does not maintain long-term support for older versions. Only the `main` branch is actively maintained.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
